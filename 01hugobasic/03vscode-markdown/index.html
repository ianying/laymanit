第三章 VS Code和Markdown

Visual Studio Code简介

优点，与Atom的对比

  • 有绿色版，无需安装
  • 文件大小大概小了一半
  • 自带terminal，不需要安装插件，直接ctrl-`可以调出。
  • tab已经默认是4个空格了，不需要再重新设置

绿色版的安装
