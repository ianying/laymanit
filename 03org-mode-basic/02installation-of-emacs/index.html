Emacs的安装和初步配置

下载

  • EMACS官方网站：https://www.gnu.org/software/emacs/
  • 根据操作系统，找到最新的版本，下载zip压缩包。

安装

  1. 无需安装。
  2. 解压后放到合适的目录，英文路径。

使用

  1. 在bin文件夹下找到runemacs.exe，建立快捷方式。
  2. 可使用addpm创建开始菜单快捷方式。

初始简单配置

配置文件

  • 配置文件是.emacs文件，如果还没有设置过emacs，这个文件尚未被创建。
  • 通常情况下，windows用户，配置文件位于用户文件夹，如: C:\Users\你的用户名\Roaming 与一个.emacs文件夹位于同一个目录下。
  • 可以通过搜索.emacs文件夹找到。
  • 如果设置了HOME环境变量，则位于设置的目录下。

设置字体和主题

尽量使用菜单栏和交互界面来设置。

设置字体并创建.emacs配置文件

可以通过调整字体来创建.emacs配置文件。方法：菜单栏 Option - Set defualt font 。设置完成确定后，需要再次点击菜单栏： Option - Save option （此时.emacs文件生成）

调整主题

菜单栏 Option - Customize Emacs - Custome Themes , 选择需要的主题，然后点击"Apply and save"。

配置文件的维护

  • 可以通过直接粘帖他人的配置到自己的.emacs文件中。
  • 如果配置完没有生效，建议重新启动Emacs，如果启动报错，应调整新添加的配置。
  • 对于每一条配置，可以自行添加备注，注释符号为： ;; 。
  • 注意经常备份.emacs文件。
